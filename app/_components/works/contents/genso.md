## 作品概要

Web3 MMORPG「元素騎士オンライン」のホームページです。
新規ユーザー増加を目的としたSEO対策を行うにあたり、従来のHTML / CSS / JavaScriptで構築されたレガシーなシステムをNext.jsで刷新しました。
上長と目指す方向性をすり合わせた上でデザイナーにデザインを発注し、自分はフロントエンド実装を担当しました。

## 工夫した点

### ポイント1：SEO対策

新規ユーザー獲得のため、以下のSEO施策を実施した。

- metaタグ最適化：title・description・OGPタグを各ページに適切に設定し、検索結果やSNSシェア時の表示を改善した
- 構造化データ：JSON-LDによるマークアップを実装し、検索エンジンにコンテンツの意味を正しく伝えるようにした
- パフォーマンス改善：画像最適化やNext.jsのSSG/SSRを活用し、Core Web Vitalsのスコアを改善した
- llms.txt：LLMがサイト情報を正しく参照できるよう、llms.txtを設置した

### ポイント2：GA4 / GTM / A/Bテストによるデータ分析基盤

Google Analytics 4とGoogle Tag Managerを導入し、ユーザー行動を計測できる基盤を構築した。
A/Bテストも実施し、データに基づいてページの改善を行った。

### ポイント3：ユーザー動画を活用した演出

ユーザーが作成したUGCコンテンツを背景素材として活用し、実際のゲームの魅力がダイレクトに伝わるようにした。
コミュニティの盛り上がりを視覚的に表現することで、新規ユーザーの興味喚起につなげた。
